●タイトル画面

　ゲームが起動すると、タイトル画面が表示されます。

　・Basic   :     30秒間プレイします。
　・Endless :     エンドレスモードでプレイします。
　・Record  :     各ステージのスコアボードを開きます。(未実装)
　・KeyCon  :     設定画面を開きます。(未実装)
　・Exit    :     ゲームを終了します。

●ルール説明

　タイトル画面から移り変わった瞬間に操作は可能です。
　ジャンプ(Wを押す)したらゲームスタートです。
　床についた時点で終了になります。

　Endlessモード
　地面に着地するまで永遠に飛べます。
　体力が追加され、0になると制御権を失います。
　スピード1000m毎に上昇します。ニードルとアイテムの効果が変わります。
　

●障害物、アイテム説明

　・(オブジェクト):(Basic説明)/(Endless説明)
　・アイテム    :　取るとスピードアップし、スコアに加算されます。/ 取ると体力が全回復します。(回復可能ﾎﾟｲﾝﾄ分)
　・ニードル    :　当たると得点が減ります。/ 当たると体力が減ります。(50-150, 1/3回復可能)

●ゲーム操作

<基本操作>
・A/D         :  左右移動
・W           :  ジャンプ
・S           :  急降下

・W/S         :  選択
・Space       :  決定


●諸注意
　
　著作権フル無視しているため、音声の半分程度は公開したら訴えられます
　リザルトを変更したまま修正していないため、Spacec連打しないとタイトルに戻れないカモ
